# Introduction to Git & GitHub

Learn how to use Git for version control and collaborate using GitHub's platform.

## Objective

Reinforce your understanding of Git and GitHub by practicing essential concepts and workflows.

## Assignment Tasks

### 1. Git Basics

1. **Installation & Setup**

   - Verify Git installation on your system
   - Install from official website if needed
   - Configure Git with your name and email

2. **Repository Creation**

   - Create a new folder for the assignment
   - Initialize it as a Git repository
   - Create a `README.md` file
   - Write a brief introduction about yourself

3. **Version Control**
   - Stage your changes
   - Create your first commit
   - Verify the commit history

### 2. Working with GitHub

1. **Repository Setup**

   - Create a new GitHub repository named `GitHub-Basics`
   - Link your local repository to GitHub
   - Push your local commits
   - Verify file appearance on GitHub

2. **Basic GitHub Operations**
   - Create a new branch
   - Make changes in the new branch
   - Create a pull request
   - Merge the changes

### 3. Collaboration Features

1. **Issues**

   - Create a new issue
   - Add labels and descriptions
   - Practice closing issues

2. **Documentation**
   - Update README.md with:
     - Project description
     - Setup instructions
     - Usage guidelines

## Testing Requirements

- Verify all Git commands work correctly
- Ensure GitHub repository is properly configured
- Check that all commits are properly pushed
- Validate branch operations and merges
