# Introduction to Git & Github

## Objective:

Reinforce your understanding of Git and GitHub by practicing essential concepts and workflows.

---

### Part 1: Git Basics

1. Verify that Git is installed on your system. If not, install it from the official website.
2. Set up your Git configuration by adding your name and email.
3. Create a new folder for this assignment and initialize it as a Git repository.
4. Inside the folder, create a file (e.g., `README.md`), write a brief introduction about yourself, and save it.
5. Stage and commit the changes in your repository.
6. Check the commit history to ensure the changes were saved.

### Part 2: Working with GitHub

1. Create a new repository on GitHub and name it `GitHub-Basics`.
2. Link your local repository to the newly created GitHub repository.
3. Push your local commits to the GitHub repository.
4. Confirm that the file appears on GitHub.
