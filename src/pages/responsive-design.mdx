# Responsive Design

### **Objective:**

- Clone the **landing page design** from the provided link with media queries to make it mobile-friendly.
- **No interactivity required** (e.g., navbar links don't need to work).
- Exclude the **lessons card** on the second image from the design.

### **Steps to Complete the Assignment**

### 1. **File Structure**

- Create the following file structure for the project:
  - `index.html`: The main HTML file.
  - `styles.css`: Contains the CSS for desktop and mobile views.
  - `images/`: A folder to store all the images and logos used in the project.

### 2. **Design Requirements**

- **Images and Logos:**
  - Use placeholder images or logos of your choice. You can source images from:
    - [Unsplash](https://unsplash.com/)
    - [Lorem Picsum](https://picsum.photos/)
    - [Freepik](https://www.freepik.com/)
- **Sections to Include:**
  - **Hero Section:**
    - Include a title, subtitle, call-to-action button(s), and a relevant background image.
  - **Features Section:**
    - Use icons or images alongside short descriptions of key features.
  - **About Section:**
    - Include a brief description of the course and its benefits, with a relevant image.
  - **Testimonials Section:**
    - Add a placeholder for user testimonials or reviews (use dummy text).
  - **Footer Section:**
    - Add simple footer content, such as copyright text and social media icons.
- **Typography and Colors:**
  - Match the font styles and colors as closely as possible to the design in the reference link.
  - Use **Google Fonts** for font families if needed.
- **Media Queries for Mobile:**
  - Ensure the layout is responsive and adapts to smaller screens (e.g., smartphones).
  - Example adjustments:
    - Stack elements vertically.
    - Reduce font sizes and padding.
    - Optimize images for smaller screens.

### 3. **Technical Notes**

- **HTML and CSS Only:**
  - Focus on replicating the **design** using pure HTML and CSS.
  - No need to implement JavaScript or functional navbar links.
- **Responsive Design:**
  - Use **media queries** for responsiveness.
  - Target breakpoints:
    - **Tablet:** 768px.
    - **Mobile:** 480px.
