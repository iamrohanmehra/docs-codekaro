# Introduction to React & Components

## **Objective:**

Build a React application to reinforce the concepts of creating and using components.

**Instructions:**

1. **Set up your project:**
   - Create a new React project using Vite.
   - Ensure the project runs successfully in your browser.
2. **Create the following components:**
   - **Header Component:** Displays the title of your app (e.g., "My React App").
   - **Main Component:** Contains a short description of what your app does or your learning journey with React.
   - **Footer Component:** Displays your name and the current year.
3. **Use the components in `App.jsx`:**
   - Import and arrange the Header, Main, and Footer components to display a complete webpage structure.
4. **Styling (Optional):**
   - Add basic CSS styling to make your app visually appealing.
