# Creating Elements

## **Objective:**

Create a simple web page that changes its background color dynamically when a button is clicked.

### **Task Instructions:**

1. **Set Up the HTML Structure:**
   - Create a web page with:
     - A heading that says **"Click the Button to Change the Background Color!"**
     - A button labeled **"Change Color"**.
2. **Background Color Change:**
   - When the user clicks the button, change the background color of the page to a random color.
   - Use JavaScript to generate random colors dynamically.

---

### **Bonus Task (Optional):**

- Display the current color's hex code (e.g., `#ff5733`) on the page.
