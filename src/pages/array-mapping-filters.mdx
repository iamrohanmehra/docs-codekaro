# Array Mapping & Filters

## **Objective:**

Utilize the provided **`movies`** array to perform various operations using **`map()`**, **`filter()`**, and method chaining. This assignment will help you understand how to manipulate and extract data from arrays effectively.

### **Dataset:**

Each movie object has the following properties:

- **`title`**: The title of the movie.
- **`genre`**: The genre(s) of the movie (comma-separated string).
- **`year`**: The year the movie was released.
- **`imdbRating`**: The IMDb rating of the movie.
- **`actors`**: An array of actors featured in the movie.

- Find all **Drama Movies** featuring **Christian Bale**.
