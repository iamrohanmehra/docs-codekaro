# React Router DOM

### **Objective:**

Create an Instagram-like user feed using React and React Router DOM. The goal is to display a list of users with their respective posts. Users can click on a specific post to view detailed information about it.

---

### **Requirements:**

1. **User Feed Page:**
   - Display a list of users.
   - Each user should have a set of posts visible below their name.
   - Posts should be displayed as images in a grid or row layout under the user's section.
2. **Post Details Page:**
   - When a user clicks on a post, navigate to a **Post Details Page** that displays:
     - The post's image.
     - The number of likes the post has.
     - The caption of the post.
     - The number of people who viewed the post.
     - The number of people who shared the post.
   - Do **not** create a pop-up for this. Use proper routing to navigate to the Post Details Page.
3. **Routing:**
   - Use `React Router DOM` to implement navigation:
     - A route for the **User Feed Page**.
     - A route for the **Post Details Page** that dynamically shows the details of the selected post.
4. **Design:**
   - Focus on functionality, not on design. Keep it simple and clean.
