# Introduction to HTML

Learn how to create web pages using fundamental HTML elements and embedding techniques.

## Objective

Practice and demonstrate your understanding of HTML elements such as headings, paragraphs, anchor tags, media tags, and embedding external content.

## Assignment Tasks

### 1. Heading

- Use the `<h1>` tag to create a **main heading** for your page
- Welcome visitors to your assignment project

### 2. Paragraph

- Use the `<p>` tag to create a **paragraph**
- Include a brief description of your assignment
- Provide information about the purpose and content

### 3. Anchor Tags

Create a list using `<ul>` and `<li>` tags with four items:

1. **Email Address**
   - Link using the `mailto:` attribute
2. **Phone Number**
   - Link using the `tel:` attribute
3. **WhatsApp Message**
   - Link using `https://wa.me/`
4. **External Website**
   - Link with `target="_blank"`

### 4. Image Tag

- Use the `<img>` tag to insert an image
- Include proper `alt` attribute
- Ensure the image is relevant to your content

### 5. Video Tag

- Embed a video using the `<video>` tag
- Include playback controls
- Support either local or online source

### 6. Audio Tag

- Add an audio file using the `<audio>` tag
- Provide playback controls
- Support either local or online source

### 7. YouTube Integration

- Embed a YouTube video using `<iframe>`
- Get the embed code from YouTube
- Ensure proper sizing and responsiveness

### 8. Instagram Integration

- Embed an Instagram post using:
  - `<blockquote>` for content
  - `<script>` for functionality
  - Official embed code from Instagram

## Testing Requirements

- Verify all links are functional
- Test media playback
- Check embedded content display
- Ensure proper loading of external resources
